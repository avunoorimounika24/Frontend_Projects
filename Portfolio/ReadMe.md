-> Personal Portfolio Website

A responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my skills, projects, education, and contact information.

 ->Features

* Responsive design for desktop, tablet, and mobile devices
* Smooth navigation between sections
* About Me section highlighting my background and interests
* Skills section showcasing technical competencies
* Projects section featuring:

  * Weather App
  * To-Do List Application
* Contact section with GitHub, LinkedIn, and email links
* Resume access for recruiters and employers

-> Technologies Used

* HTML5
* CSS3
* Git & GitHub

->Purpose

This portfolio serves as a central place to showcase my frontend development skills, projects, and learning journey as a Computer Science and Engineering student. It is designed to provide recruiters, internship providers, and collaborators with an overview of my technical abilities and project experience.
